## 🎮 Présentation & Concept

**Blacksnack** réinvente le grand classique "Snake" en y ajoutant une dimension compétitive et financière. 

### Pourquoi jouer sur Blacksnack ?
- **Adrénaline** : Jouez avec de véritables enjeux.
- **Accessibilité** : Rechargez et retirez via **Mobile Money** (M-Pesa, Airtel Money, Orange Money).
- **Équité** : Tout le monde joue avec les mêmes règles, seul le talent compte.
- **Communauté** : Défiez d'autres joueurs en direct (1v1).

---

## 🕹️ Comment Jouer (Les Bases)

### 1. Contrôles & Maniabilité
Sur **Mobile**, vous avez deux options pour diriger votre serpent :
- **Touches Tactiles** : Utilisez les flèches directionnelles sous la zone de jeu.
- **Gestes (Swipe)** : Glissez votre doigt sur l'écran dans la direction souhaitée.

### 2. Règles d'Or
- 🍎 **Mangez les proies** : Chaque proie vous rapporte **+1 point** et fait grandir votre serpent.
- ⚡ **Évitez les murs** : Toucher la bordure du terrain signifie la fin de la partie.
- 🐍 **Ne vous mordez pas** : Évitez de percuter votre propre corps.
- ⏱️ **Vitesse** : Plus vous allez vite, plus vos gains sont multipliés !

### 3. Les Vitesses de Jeu
Choisissez votre défi dans les paramètres (Menu > Autres > Vitesse) :

| Mode | Vitesse | Gain | Pour qui ? |
|:---:|:---:|:---:|:---:|
| 🐌 **Lent** | 200ms | **x0.8** | Débutants complets |
| 🚶 **Normal** | 150ms | **x1.0** | Le standard Blacksnack |
| 🏃 **Rapide** | 100ms | **x1.2** | Joueurs confirmés |
| ⚡ **Extrême** | 75ms | **x1.5** | Les maîtres du serpent |

## 🎯 Modes de Jeu

---

### 2. 🏆 Mode Tournoi

**Localisation :** Menu → Tournois

**Comment ça fonctionne :**

#### Inscription à un Tournoi
1. Consulter la liste des tournois actifs
2. Voir les informations :
   - 💰 **Cagnotte totale** (Prize Pool)
   - 🎫 **Frais d'entrée** : 1 USD ou 1000 CDF
   - 👥 **Nombre de participants** : X/MAX
   - ⏰ **Temps restant** avant la fin
3. Cliquer sur "Rejoindre le tournoi"
4. Choisir la devise (USD ou CDF)
5. Les frais sont déduits du solde
6. ✅ **Inscription confirmée !**

#### Déroulement du Tournoi
1. Une fois inscrit, accéder au jeu via "Jouer"
2. L'interface affiche :
   - 🎮 Zone de jeu Snake
   - 📊 Classement en temps réel à droite
   - 🏅 Votre position dans le classement
3. Jouer autant de fois que souhaité
4. **Seul le meilleur score compte**
5. Scores mis à jour en temps réel

#### Fin du Tournoi et Gains
Quand le tournoi se termine (date limite atteinte) :

**Distribution des prix (exemple pour 100 participants) :**
- 🥇 **1er place** : 40% de la cagnotte
- 🥈 **2e place** : 25% de la cagnotte
- 🥉 **3e place** : 15% de la cagnotte
- 🏅 **4e-10e** : 10% partagés
- 🎖️ **11e-20e** : 10% partagés

**Calcul automatique :**
```
Cagnotte = (Frais d'entrée × Nombre de participants)
Prize 1er = Cagnotte × 0.40
Prize 2e = Cagnotte × 0.25
...
```

**Les gains sont automatiquement crédités sur le solde du joueur ! 💰**

---

### 3. ⚔️ Mode Défi (Challenge)

**Localisation :** Menu → Défis

**Comment ça fonctionne :**

#### Créer un Défi
1. Cliquer sur "Créer un défi"
2. Choisir la mise : **Entre 20$ et 400$**
3. Confirmer (l'argent est mis en jeu)
4. Le défi apparaît dans la liste publique
5. Attendre qu'un adversaire accepte

#### Accepter un Défi
1. Consulter la liste des défis disponibles
2. Voir :
   - 👤 Nom du créateur
   - 💰 Mise en jeu
3. Cliquer sur "Accepter"
4. La même somme est déduite de votre solde
5. ✅ **Le défi commence !**

#### Déroulement du Défi
1. **Chaque joueur joue de son côté**
2. L'interface montre :
   - Votre score
   - Score de l'adversaire (si déjà joué)
   - Statut du défi
3. Chacun a **une seule tentative**
4. Pas de limite de temps pour jouer

#### Résultat du Défi
Quand les deux joueurs ont joué :

**Le joueur avec le MEILLEUR score gagne !**

```
Mise totale = (Mise créateur + Mise adversaire)
Gain du gagnant = Mise totale (double de sa mise)
```

**Exemple :**
- Défi créé : 50$
- Adversaire accepte : 50$
- Cagnotte totale : 100$
- Score créateur : 35 points
- Score adversaire : 42 points
- 🏆 **Adversaire gagne 100$** (profit net : 50$)
- 😢 Créateur perd sa mise de 50$

**Les gains sont automatiquement crédités ! 💰**

## 💰 Système de Gains

### Comment Gagner de l'Argent

#### 1. Via les Tournois 🏆
- Terminer dans le TOP du classement
- Plus il y a de participants, plus la cagnotte est élevée
- Gains automatiques à la fin du tournoi
- Historique visible dans "Revenus"

#### 2. Via les Défis ⚔️
- Battre un adversaire en 1v1
- Gagner le double de sa mise
- Risque = Récompense élevée
- Gains immédiats après victoire

#### 3. Multiplicateur de Vitesse
Jouer en vitesse **Rapide** ou **Extrême** augmente les gains potentiels grâce au multiplicateur !

### Historique des Revenus

**Localisation :** Menu → Profil → Revenus

Affiche :
- 📅 Date et heure
- 💰 Montant gagné
- 📝 Source (Tournoi X / Défi contre Y)
- 💵 Total des gains cumulés

## 💳 Système de Portefeuille

### Dépôt d'Argent

#### Via Mobile Money
1. Menu → Portefeuille → Dépôt
2. Entrer le montant (USD ou CDF)
3. Choisir "Mobile Money"
4. Cliquer sur "Déposer"
5. **Modal s'ouvre :**
   - Entrer numéro de téléphone
   - Choisir réseau : Vodacom / Airtel / Orange
   - Confirmer le montant
6. Cliquer sur "Procéder"
7. ⏳ Traitement (2 secondes)
8. ✅ **Solde mis à jour !**

#### Via Carte Visa
1. Menu → Portefeuille → Dépôt
2. Entrer le montant (USD ou CDF)
3. Choisir "Visa"
4. Cliquer sur "Déposer"
5. **Modal s'ouvre :**
   - Numéro de carte (16 chiffres)
   - Nom sur la carte
   - Date d'expiration (MM/AA)
   - CVV (3-4 chiffres)
   - Confirmer le montant
6. Cliquer sur "Procéder"
7. ⏳ Traitement (2 secondes)
8. ✅ **Solde mis à jour !**

### Retrait d'Argent

**Même processus que le dépôt :**
- Vérification du solde disponible
- Choix de la méthode de paiement
- Confirmation des informations
- Traitement instantané

### Taux de Change
```
1 USD = 2500 CDF
```
Conversion automatique dans l'interface.
